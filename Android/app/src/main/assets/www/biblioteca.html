Sobre Eclipse

Eclipse é um sistema de RPG moderno com inspiração em horror tecno-paranormal, ambientado em um mundo onde o Obscuro atravessa a realidade cotidiana. Agentes investigam, combatem e cedem ao Outro Lado enquanto o sistema pesa escolhas contra consequências.

Filosofia de Design

  • Nomes completos. Vitalidade, Esforço, Foco e Determinação no lugar de siglas frias
  • Ordem de leitura. Do fundamental ao avançado, do jogador ao mestre
  • Conteúdo cru. Dados brutos no código, polimento aplicado no momento da leitura
  • Interface leve. Grid aesthetic 220×220, hover revelando informação, sem ruído visual

Estrutura Técnica

Arquivo único HTML com CSS e JavaScript embedados. Capítulos armazenados em um bloco JSON no próprio documento, processados pela função cleanContent antes da renderização. Pronto para futura migração para interface dinâmica sem alterar os dados fonte.

Notas de Uso

  • Clique em qualquer card para abrir a leitura completa
  • Use as setas ◀ ▶ no topo do leitor para navegar entre capítulos
  • O botão Sortear aparece só em Complicações e destaca um traço aleatório
  • A seta ↑ no canto inferior volta ao topo da página ou do capítulo
  • Esc fecha o leitor ou este painel
  • A barra de busca filtra capítulos por título em tempo real

Convenções do Sistema

  • Dificuldade 10 tarefa simples, 14 comum, 18 difícil, 24 desafiador, 28 extremo, 30+ lendário
  • Dados de dano crescem em passos: 1d4 → 1d6 → 1d8 → 1d10 → 1d12 → 2d8...
  • Modificadores de dano são expressos como multiplicadores (×1,2, ×1,4, ×1,8)
  • Valores fracionários usam palavras (metade, um quarto, dois quintos) em vez de porcentagem
